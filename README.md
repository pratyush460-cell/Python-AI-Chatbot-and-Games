# 🤖 Python-AI-Chatbot-and-Games

A collection of Python-based AI projects including an automated WhatsApp chatbot, Gemini API integration, and interactive games.

---

## 🚀 Projects

### 🤖 AI Auto-Reply Chatbot (`chatbot/`)
An automated WhatsApp chatbot that reads chat history using PyAutoGUI and generates AI-powered replies using **Google Gemini API**.

**Features:**
- Auto-detects new messages from a specific sender
- Generates context-aware replies using Gemini AI
- Auto-types and sends the response via PyAutoGUI

**Tech Used:** Python, Google Gemini API, PyAutoGUI, Pyperclip

---

### 🎯 Perfect Guess Game (`perfect_guess/`)
A number guessing game where the player tries to guess a randomly generated number.

**Tech Used:** Python

---

### ✂️ Snake Water Gun (`snake_water_gun/`)
A Python implementation of the classic Snake-Water-Gun game (similar to Rock-Paper-Scissors).

**Tech Used:** Python

## ⚙️ Installation

```bash
# Clone the repo
git clone https://github.com/pratyush460-cell/Python-AI-Chatbot-and-Games.git

# Install dependencies
pip install google-generativeai pyautogui pyperclip
```

---

## 🔑 Setup Gemini API Key

1. Go to [aistudio.google.com](https://aistudio.google.com)
2. Generate your free API key
3. Replace `YOUR_GEMINI_API_KEY_HERE` in `chatbot/gemini_api.py`
